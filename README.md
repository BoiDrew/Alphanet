# 🧹 Cleaning Supply Request Tracker

A browser-based supply request and analytics system for cleaning teams.

## Features

- Request lifecycle tracking
- Urgent request management
- Analytics dashboard
- CSV export
- Job-site filtering
- Persistent local storage

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Chart.js
- Vercel Deployment
